Welcome to OptiCrop

Smart Crop Recommendation System

OptiCrop is a machine learning-based crop recommendation system designed to help users identify the most suitable crop for given soil and environmental conditions.

The system analyzes important agricultural factors such as nitrogen, phosphorus, potassium, temperature, humidity, soil pH and rainfall to recommend a suitable crop.

By using a trained machine learning model, OptiCrop aims to support better crop selection and help users make data-driven agricultural decisions.
